# مدد — دليل التشغيل الكامل

موقع مكتمل: بحث ومقارنة أسعار جملة، حسابات موردين/مشترين، طلبات عروض أسعار،
تقييمات، صفحات موردين، مفضّلة، صور منتجات، تنبيهات (واتساب + داخل الموقع)، ولوحة إدارة.

## 1) شغّل الـ migrations بالترتيب
من Supabase → SQL Editor (أو `supabase db push`)، نفّذ ملفات `supabase/migrations/` بالترتيب الزمني:
1. ملفّا Lovable الأصليان (`...130629`, `...130645`) — موجودان أصلاً في مشروعك.
2. `20260608140000_seed_demo_data.sql` — بيانات تجريبية.
3. `20260608150000_favorites_images_quotes.sql` — مفضّلة + صور + طلبات عروض.
4. `20260608170000_in_app_notifications.sql` — إشعارات داخل الموقع.
5. `20260608180000_add_admin_role.sql` — إضافة دور admin.
6. `20260608180001_admin_policies.sql` — صلاحيات الأدمن.
7. `20260608190000_reviews.sql` — التقييمات.
8. (اختياري لواتساب) `20260608160000_quote_notify_webhook.sql` — بعد نشر الدالة (راجع `واتساب_التنبيهات_خطوات.md`).

## 2) عيّن نفسك أدمن (بعد التسجيل في الموقع)
```sql
INSERT INTO public.user_roles (user_id, role)
SELECT id, 'admin' FROM auth.users WHERE email = 'بريدك@مثال.com'
ON CONFLICT (user_id, role) DO NOTHING;
```

## 3) شغّل الموقع
```bash
npm install --legacy-peer-deps
npm run dev
```

## الصفحات
| المسار | الوصف | يتطلب دخول |
|---|---|---|
| `/` | الرئيسية | لا |
| `/search` | بحث ومقارنة (فلترة فئة/مدينة، ترتيب) | لا |
| `/product/$id` | مقارنة عروض المنتج + طلب عرض سعر | لا |
| `/supplier/$id` | صفحة المورّد + منتجاته + تقييماته | لا |
| `/auth` | تسجيل/دخول | لا |
| `/dashboard` | لوحة المورّد (إضافة/تعديل منتجات + طلبات واردة) | نعم |
| `/favorites` | المفضّلة | نعم |
| `/my-requests` | طلباتي (المشتري) | نعم |
| `/account` | إعدادات الحساب | نعم |
| `/admin` | لوحة الإدارة | نعم (admin) |

## قائمة المميزات
- ✅ بحث وفلترة (فئة + مدينة) وترتيب، مع أرخص/متوسط/أعلى سعر لكل منتج.
- ✅ صفحة منتج بكل عروض الموردين مرتّبة + أزرار اتصال/واتساب + طلب عرض سعر.
- ✅ صفحة مورّد عامة + قائمة منتجاته + تقييمات (نجوم + تعليق، متوسط تلقائي).
- ✅ مفضّلة، صور منتجات (Storage)، تعديل/حذف أسعار العروض.
- ✅ طلبات عروض أسعار (RFQ) مع حالات ورد المورّد.
- ✅ تنبيهات: واتساب (Edge Function) + داخل الموقع (جرس + سجل).
- ✅ لوحة إدارة: إحصائيات + إدارة موردين (توثيق/حذف) ومنتجات (حذف).

تم التحقق: `tsc --noEmit` ✅ و `npm run build` ✅.
