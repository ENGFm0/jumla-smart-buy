# تنبيهات واتساب لمنصة "مدد" — دليل التجهيز

التنبيه يُرسَل تلقائياً عبر هذا المسار:
**قاعدة البيانات (طلب جديد/رد)** ← **مُشغّل** ← **Edge Function `quote-notify`** ← **WhatsApp Cloud API**.

الكود جاهز (الدالة + المُشغّل). يتبقّى تجهيز حساب واتساب والأسرار. اتبع الخطوات بالترتيب.

---

## 1) جهّز WhatsApp Cloud API (مرة واحدة)
1. ادخل [Meta for Developers](https://developers.facebook.com) → أنشئ تطبيقاً نوع **Business**.
2. أضف منتج **WhatsApp**. ستحصل على:
   - **Phone Number ID** (معرّف الرقم) — انسخه.
   - **Temporary access token** للتجربة (ساعات)، ولاحقاً أنشئ **توكن دائم** عبر System User.
3. للتجربة: أضف رقم جوالك في قائمة **أرقام الاختبار** (To). للإطلاق الحقيقي: وثّق رقم العمل وفعّل الحساب.

## 2) أنشئ قالبين معتمدين (Message Templates)
من **WhatsApp Manager → Message Templates**، أنشئ قالبين (الفئة: **Utility**، اللغة: **العربية**):

**القالب الأول — اسمه `madad_new_quote`** (إشعار المورّد):
```
مرحباً {{1}}، وصلك طلب عرض سعر جديد على منتج "{{2}}" بكمية {{3}} عبر منصة مدد. ادخل لوحتك للرد على الطلب.
```
**القالب الثاني — اسمه `madad_quote_reply`** (إشعار المشتري):
```
بخصوص طلبك لمنتج "{{1}}" في منصة مدد: {{2}}. التفاصيل في صفحة «طلباتي».
```
> انتظر حتى تتحول حالة القالبين إلى **Approved** (عادة دقائق إلى ساعات).

## 3) انشر الدالة وأضف الأسرار
عبر Supabase CLI (بعد `supabase link` لمشروعك):
```bash
supabase functions deploy quote-notify --no-verify-jwt

supabase secrets set \
  WHATSAPP_TOKEN="توكن_ميتا" \
  WHATSAPP_PHONE_NUMBER_ID="معرّف_الرقم" \
  WHATSAPP_TEMPLATE_SUPPLIER="madad_new_quote" \
  WHATSAPP_TEMPLATE_BUYER="madad_quote_reply" \
  WHATSAPP_LANG="ar" \
  WEBHOOK_SECRET="اختر_سر_عشوائي_قوي"
```
(`SUPABASE_URL` و `SUPABASE_SERVICE_ROLE_KEY` متوفّران تلقائياً للدالة.)

## 4) اربط المُشغّل بالدالة (طريقتان — اختر واحدة)

**الطريقة (أ) — لوحة Supabase (الأسهل):**
Database → **Webhooks** → Create:
- الجدول: `quote_requests`، الأحداث: **Insert** و **Update**.
- النوع: **Supabase Edge Functions** → اختر `quote-notify`.
- أضف ترويسة: `x-webhook-secret` = نفس `WEBHOOK_SECRET`.

**الطريقة (ب) — SQL:**
شغّل `supabase/migrations/20260608160000_quote_notify_webhook.sql` بعد استبدال
`YOUR_PROJECT_REF` و `YOUR_WEBHOOK_SECRET` بقيمك.

## 5) جرّب
- سجّل دخول كصاحب محل وأرسل «طلب عرض سعر» → المورّد يستقبل رسالة واتساب.
- المورّد يردّ بسعر من لوحته → المشتري يستقبل رسالة واتساب.

---

## ملاحظات مهمة
- **أرقام المستلمين:** يُؤخذ رقم المورّد من حقل `whatsapp`، ورقم المشتري من `phone` في ملفه.
  تأكد أنها أرقام واتساب صحيحة (تُحوَّل تلقائياً لصيغة 9665XXXXXXXX).
- **بيانات تجريبية:** الموردون التجريبيون أرقامهم وهمية، فلن تصل لهم رسائل فعلية — جرّب بحساب مورّد حقيقي برقمك.
- **التكلفة:** رسالة Utility في السعودية ≈ 0.04–0.06 ر.س للرسالة (ردودك خلال ٢٤ ساعة من تواصل المستخدم مجانية).
- لم يُختبر هذا جزء الدالة محلياً (يحتاج بيئة Deno/Supabase)، لكنه مكتوب وفق أصول Supabase Edge Functions.
